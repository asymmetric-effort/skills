# Security Skills

Auditing, hardening, vulnerability analysis, and secure coding practices.

## Subclasses

| Subclass                 | Description                                        |
|--------------------------|----------------------------------------------------|
| `auditing/`              | Code and infrastructure security audits            |
| `cryptography/`          | Encryption, hashing, key management, certificates  |
| `hardening/`             | System and application hardening techniques        |
| `incident-response/`     | Triage, forensics, and remediation workflows       |
| `pentest/`               | White-box penetration test code review             |
| `secure-coding/`         | Defensive coding patterns and input validation     |
| `vulnerability-analysis/`| CVE analysis, dependency scanning, exploit review  |
