# Data Skills

Data processing, analysis, transformation, and visualization.

## Subclasses

| Subclass            | Description                                     |
|---------------------|-------------------------------------------------|
| `analysis/`         | Statistical analysis, exploration, insights     |
| `pipelines/`        | ETL, data workflows, streaming                  |
| `processing/`       | Parsing, cleaning, normalization                |
| `scraping/`         | Web scraping and data extraction                |
| `transformation/`   | Format conversion, reshaping, enrichment        |
| `visualization/`    | Charts, graphs, dashboards                      |
