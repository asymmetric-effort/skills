# DevOps Skills

CI/CD pipelines, containers, infrastructure, deployment, and monitoring.

## Subclasses

| Subclass           | Description                                     |
|--------------------|-------------------------------------------------|
| `ci-cd/`           | Pipeline configuration, build automation        |
| `containers/`      | Docker, Kubernetes, orchestration               |
| `deployment/`      | Release strategies, rollbacks, blue-green       |
| `infrastructure/`  | IaC, Terraform, cloud provisioning              |
| `monitoring/`      | Observability, alerting, logging infrastructure |
| `networking/`      | DNS, load balancing, firewalls, VPNs            |
